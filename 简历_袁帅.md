## 项目经验

### 会议纪要智能体（RAG 知识库问答系统）

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;基于 Spring Boot + pgvector + DeepSeek + AgentScope Harness 自研的 RAG 知识库问答系统，支持文档上传、自动 ETL 切分入库、混合检索以及流式 Agent 对话，实现会议纪要场景下的智能问答。

- **工作内容**：
  - **双通道混合检索**：同时整合 pgvector 向量检索和 PostgreSQL tsvector 全文检索，使用 RRF（Reciprocal Rank Fusion）算法融合排序，显著提升语义和关键词检索的互补效果。
  - **结构化感知切分**：针对会议纪要场景，按发言人轮次、总结/决策标记自动分段，配合 overlap 窗口和句子边界智能拆分，平衡语义完整性和检索粒度。
  - **检索质量保障**：设计四级证据约束（NONE/WEAK/PARTIAL/SUFFICIENT）动态控制 Agent 回答策略；低于置信度阈值时自动改写 query 二次检索，结合时间衰减加权和 MMR 多样性去重，提升召回精准度。
  - **AI 对话集成**：将 RAG 检索链路嵌入 AgentScope HarnessAgent 流式对话，采用预检索 + 上下文注入策略，配合 SearchGuard 防重复检索，统一知识库问答与普通闲聊入口。
  - **全链路参数可配置**：通过 @ConfigurationProperties 集中管理 20+ 检索参数（chunk 大小、TopK、证据阈值、时间衰减权重、rerank 开关等），支持离线评估和运行态调优。

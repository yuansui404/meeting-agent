## 项目经验

### 会议纪要智能体（RAG 知识库问答系统）

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;基于 Spring Boot + pgvector + DeepSeek 自研的 RAG 知识库问答系统，支持文档 ETL、混合检索、知识库风格改写与流式 Agent 对话。

- **混合检索**：会议纪要语义分散，纯向量或全文检索效果有限。整合 pgvector 向量检索 + PostgreSQL tsvector 全文检索，设计 RRF 融合排序与结构化感知切分（发言人轮次/决策标记/overlap 窗口），Top-5 召回率提升 30%+。
- **知识库风格改写**：纪要改写需保持风格统一，手动指定模板效率低。改写时自动从知识库检索语义相似文档作为风格范例注入 Prompt，支持人工星标文档优先排序 + 「换一种风格」排除已用引用，实现自适应的智能风格改写。
- **反馈驱动优化**：段落级点赞/点踩实时调整向量块 priority_score（±0.5），影响后续风格排序权重，形成用户反馈驱动的持续优化闭环。
- **检索质量保障**：四级证据约束动态控制回答策略，低置信度自动二次检索 + MMR 多样性去重；SSE 断连后后台自动完成结果持久化与对照 docx 生成，保障零数据丢失。
